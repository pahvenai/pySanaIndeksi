	Käyttöohje.pdf, Määrittelydokumentti.pdf,
	Testausdokumentti.pdf, Toteutusdokumentti.pdf
Material/
	Työn testaukseen käytetyt testimateriaalit.
nbproject/

src/
	Lähdekoodi kolmessa Python-paketissa (kansiossa):
	Support, Trees, WordIndex
